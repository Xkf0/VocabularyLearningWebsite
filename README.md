# 艾宾浩斯记忆助手

基于艾宾浩斯遗忘曲线的多学科复习工具。支持单词、句子、数学知识点、题目、解题方法等多种内容的科学复习。

## 功能

- **科学复习** — 13 级复习间隔（20分钟 → 1年），之后每年一次，永不停止
- **多类型支持** — 单词、句子、数学知识点、题目（含图片）、解题方法，统一纳入艾宾浩斯复习队列
- **自动音标** — 添加单词时自动从词典 API 获取音标和发音音频
- **点击发音** — 复习时播放单词发音
- **AI 造句** — 积累单词后自动调用 DeepSeek API 生成英文句子
- **图片支持** — 题目和解答支持拍照/上传图片，自动压缩后存储
- **多用户登录** — 支持服务端多用户隔离，数据按用户存储
- **云同步** — 登录后数据自动同步到服务器，多端共享
- **导出导入** — 支持完整数据备份和恢复
- **统计追踪** — 每日复习数、新增数统计

## 使用方法

### 本地模式

1. 确保电脑已安装 [Python](https://www.python.org/downloads/)
2. 双击 **`启动.bat`**，浏览器自动打开
3. 数据保存在 `vocabulary-data.json`

或直接双击 **`index.html`**（数据保存在浏览器 localStorage 中）。

### 服务器模式

1. 将项目部署到服务器，运行 `python3 server.py <端口>`
2. 浏览器访问 `http://服务器IP:端口`
3. 首次使用需注册账号，之后登录使用
4. 数据按用户隔离，自动同步到服务器

## 数据备份

### 服务端自动备份（生产环境）

服务器每日 0:00 自动备份所有用户数据到 `/var/backups/vocab/`，仅保留前一天的数据。

### 客户端手动备份

点击页面上方 **导出** 按钮，将完整数据保存为 JSON 文件。

## 技术说明

- 纯前端：HTML + CSS + JavaScript（单个 `index.html`）
- 后端：`server.py`（Python HTTP 服务，支持多用户登录和数据持久化）
- 音标数据来源：[Free Dictionary API](https://dictionaryapi.dev/)
- 存储：本地模式用 localStorage / JSON 文件，服务器模式用服务端文件存储

## AI 句子生成

每积累 10 个单词后，系统自动调用 DeepSeek API 生成包含这些单词的英文句子（附中文翻译），句子进入艾宾浩斯复习队列。

1. 点击页面顶部 **设置** 按钮
2. 在"AI 设置"中输入你的 DeepSeek API Key 并保存
3. 正常添加单词即可，每满 10 个自动触发

> API Key 仅保存在浏览器本地，不会上传到服务器。
