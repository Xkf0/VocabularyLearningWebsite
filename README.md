# 艾宾浩斯背单词

基于艾宾浩斯遗忘曲线的单词复习工具。打开即用，无需注册，数据存本地。

## 功能

- **科学复习** — 13 级复习间隔（20分钟 → 1年），之后每年复习一次，永不停止
- **自动音标** — 添加单词时自动从词典 API 获取音标和发音音频
- **点击发音** — 复习时点击 🔊 按钮播放单词发音
- **AI 造句** — 每积累 10 个单词，自动调用 DeepSeek API 生成包含这些单词的英文句子（含中文翻译），句子也加入艾宾浩斯复习队列
- **本地存储** — 数据保存在浏览器或本地 JSON 文件中，不经过任何服务器
- **导入导出** — 支持数据备份和恢复

## 使用方法

### 方法一：双击启动（推荐）

1. 确保电脑已安装 [Python](https://www.python.org/downloads/)
2. 双击 **`启动.bat`**
3. 浏览器自动打开，数据自动保存到 `vocabulary-data.json`

### 方法二：直接打开浏览器

1. 双击 **`index.html`**
2. 数据保存在浏览器中，关闭网页再打开不会丢失
3. 建议定期点击「保存到文件」手动备份

## 技术说明

- 纯前端：HTML + CSS + JavaScript（单个 `index.html` 文件）
- 后端可选：`server.py` 提供 HTTP 服务，实现自动读写 `vocabulary-data.json`
- 音标数据来源：[Free Dictionary API](https://dictionaryapi.dev/)
- 无需安装任何依赖，无需联网（除首次查询音标和 AI 造句外）

## AI 句子生成

每当你积累 10 个单词后，系统会自动调用 DeepSeek API 将这 10 个单词组合成一个通顺地道的英文句子（附中文翻译），生成的句子会像单词一样进入艾宾浩斯复习队列。

### 使用步骤

1. 点击页面顶部 **设置** 按钮
2. 在"AI 设置"中输入你的 DeepSeek API Key 并保存
3. 正常添加单词即可，每满 10 个自动触发造句

> 提示：无 API Key 时不影响单词学习功能，只是不会生成句子。API Key 仅保存在你的浏览器本地，不会上传到任何服务器。

### 句子复习

- 复习卡片顶部显示 **📝 句子 #1-10** 标识
- 显示英文句子 → 回想中文翻译
- 使用与单词相同的艾宾浩斯复习间隔
- 在「单词本」页面可查看所有已生成的句子列表

### 数据格式

导出 JSON 时，句子数据会随单词一起导出，格式为 `{words: [...], sentences: [...]}`。旧版纯数组格式仍可正常导入。
